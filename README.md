# Ads Manager Simulator

Game edukasi untuk belajar Meta Ads. Pemain jadi Junior Media Buyer,
mengelola campaign untuk klien real-world, dan menghadapi reaksi bos
sesuai ROAS yang dicapai.

## Fitur

- 5 skenario klien dari niche berbeda (kuliner, edukasi, jasa, fashion, medical)
- UI mirip Meta Ads Manager asli
- Engine simulasi realistis (audience match, creative match, dll)
- Dashboard hasil lengkap dengan reaksi bos
- Post-mortem edukasi untuk tiap keputusan
- Progress tersimpan otomatis di browser

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- lucide-react (ikon)

## Cara Deploy (untuk non-coder)

1. Upload folder ini ke GitHub (via web UI, tidak perlu Git CLI)
2. Sign up di vercel.com dengan akun GitHub
3. Import repository ini → Vercel deteksi otomatis Vite → klik Deploy
4. Dapat URL public, siap di-share
